Portfolio - Argentina Programa #YoProgramo

Desarrollo de una aplicación web full stack, que muestra: datos personales, estudios cursados, experiencia laboral, conocimiento de las tecnologías, entre otros. Esta aplicación es de arquitectura distribuida y contiene un diseño de interfaz de usuario (FrontEnd), una base de datos y las APIs necesarias para proveer a través de internet la información (BackEnd). Tecnologías Utilizadas: HTML, CSS, JavaScript - Angular - Java + Spring Boot (API Rest) - MySQL.
